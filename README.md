# 25マス文字式計算プリント

中学生向けに、文字式の加法プリントと英語アクティビティプリントを生成するWebアプリです。

## Features

- 縦横にランダムな1次式を表示
- 25マスの足し算プリントを生成
- 係数・定数の範囲を調整
- 正の数だけ、正負ミックス、負の数多めを選択
- 解答つき印刷のオン/オフ
- 合言葉による同じ問題の再生成
- 中学生英語向けの「25ライト英作文」「書きかえミッション」「単語・熟語ミッション」
- 英語は短い記入欄と語群を使い、時制・助動詞・不定詞・比較・受け身・現在完了などを混ぜて出題
- 中1前半、中1全体、中2まで、中3までの範囲切り替え

## Usage

`index.html` をブラウザで開くだけで使えます。
